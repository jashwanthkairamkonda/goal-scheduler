# Goal Scheduler

A smart web-based goal scheduler for managing tasks, tracking progress, and improving productivity.

## Tech Stack
- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth/database)

## Getting Started (Local)

### 1) Clone
```sh
git clone https://github.com/jashwanthkairamkonda/goal-scheduler.git
cd goal-scheduler
```

### 2) Install dependencies
> Use **one** package manager (recommended: npm).
```sh
npm install
```

### 3) Configure environment variables
Create a `.env` file (do **not** commit it). Use `.env.example` as a template.

### 4) Run the app
```sh
npm run dev
```

## Scripts
```sh
npm run dev
npm run build
npm run preview
npm run lint
```

## Deployment
Build the project and deploy the `dist/` output to any static hosting provider (Vercel, Netlify, Cloudflare Pages, etc.).
